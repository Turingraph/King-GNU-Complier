## User tutorial (UNFINISHED)

1.	`./fdf your_file.fdf` will convert your file (that contains isometric projection related data) in to 3D model.
2.	`A`, `S`, `D`, and `W` keys control the rotation of first light.
3.	`a`, `s`, `d`, and `w` keys control the rotation of second light.
4.	`left`, `right`, `down`, and `up` keys control the rotation of the 3D object.
5.	`./fdf "hello world"` compute the 3D isomorphic projection of the text `hello world`
